DatabaseHelper

DatabaseHelper es una utilería que permite la creación de entidades Hibernate a partir del diagrama ENTIDAD-RELACIÓN.

Reduce los errores y contratiempos que surgen durante la creación de entidades. Se emplea una herramienta colaborativa en tiempo real, permitiendo a diferentes personas diseñar conjuntamente una base de datos.

Tecnologías:

Java 100%
Spring Boot 100%
DrawIO 100%

Propuesta y desarrollo: Miguel Villa

Año: 2019

Cliente: * Requerimiento propio

ProjectManager

ProjectManager es una aplicación que gestiona de forma básica un proyecto de software; su información, estatus y demás datos generales. Adicionalmente implementa un módulo de web scraping para desplegar aplicaciones a través de Jenkins.

Tecnologías:

NodeJS 80%
Angular 10 20%

Propuesta y desarrollo: Miguel Villa

Año: 2020

Cliente: * Requerimiento propio

DevsUtilities

DevsUtilities es un proyecto que pretende agilizar el desarrollo de aplicaciones genéricas, con código repetitivo o lógica básica, generando código Java.

No pretende generar proyectos listos para su uso, sino en liberar al programador de todo trabajo que no necesite ser supervisado, permitiendo que este se enfoque en secciones del proyecto de mayor complejidad.

Tecnologías:

Java 100%
Maven 100%

Propuesta y desarrollo: Miguel Villa

Año: 2020

Cliente: cliente no disponible

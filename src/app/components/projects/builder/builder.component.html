Builder

Builder se enfoca en optimizar la codificación de proyectos basados en Java y ofrece soporte para Spring Boot, Spring Data JPA, etc., en Java.

Genera un proyecto Maven con entidades, repositorios, servicios, controladores, archivos de configuración y una interfaz básica con Thymeleaf.

Tecnologías:

Java 100%
Spring Boot 100%
Angular 10 20%

Propuesta y desarrollo: Miguel Villa

Año: 2021

Cliente: * Requerimiento propio
